# Comunidad AGI

## Descripción

Comunidad AGI es una plataforma dedicada a conectar a entusiastas y profesionales de la Inteligencia Artificial en Latinoamérica. Facilitamos el intercambio de conocimientos y la colaboración a través de podcasts, artículos y eventos.

## Tecnologías

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ShadCN UI](https://ui.shadcn.com/)
- [Radix UI](https://www.radix-ui.com/)

## Instalación

1. **Clona el repositorio:**

    ```bash
    git clone https://github.com/tu-usuario/comunidad-agi.git
    ```

2. **Instala las dependencias:**

    ```bash
    cd comunidad-agi
    npm install
    ```

3. **Inicia el servidor de desarrollo:**

    ```bash
    npm run dev
    ```

4. **Abre la aplicación:**

    Visita [http://localhost:3000](http://localhost:3000) en tu navegador.

## Uso

- **Reproductor de Audio:** Escucha podcasts directamente en la plataforma.
- **Temas:** Alterna entre modo oscuro y claro desde la configuración.
- **Navegación:** Accede a diferentes secciones como podcasts y contactos desde el menú principal.

## Contribución

¡Las contribuciones son bienvenidas! Sigue estos pasos:

1. Haz un fork del repositorio.
2. Crea una rama para tu feature:
    ```bash
    git checkout -b feature/nueva-funcionalidad
    ```
3. Realiza tus cambios y haz commit:
    ```bash
    git commit -m "Añadir nueva funcionalidad"
    ```
4. Empuja la rama:
    ```bash
    git push origin feature/nueva-funcionalidad
    ```
5. Abre un Pull Request.
